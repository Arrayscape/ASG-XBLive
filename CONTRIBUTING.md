# Contributing to ASG-XBLive

ASG-XBLive is an open source project.

It is the work of contributors like you. We appreciate your help!

# Contributing 

Please join the [asg-xblive mailing list](https://groups.google.com/a/arrayscape.com/g/asg-xblive)

Please read the Contribution Guidelines in
[CONTRIBUTING.md](CONTRIBUTING.md) before sending patches.

Unless otherwise noted, the ASG-XBLive source files are distributed
under the license found in the LICENSE file.

## How to Contribute

To contribute to this project, you must sign a Contributor License
Agreement which legally confirms you own the Copyright to the
contribution, and assigns us the rights to it.  Please reach out
to [dev-cla@arrayscape.com](mailto:dev-cla@arrayscape.com) for the
agreement.


# Filing issues

## Before filing an issue

If you are unsure whether you have found a bug, please consider
generating a minimal testcase first.

## Security Issues

Sensitive security-related issues should be reported to
[security@arrayscape.com](mailto:security@arrayscape.com).

## Non-Security Issues

The recommended way to file an issue is by recording it in the
[ASG-XBLive Issue Tracker](https://github.com/Arrayscape/ASG-XBLive/issues)
Otherwise, when filing an issue, make sure to answer these five
questions:

1. What version of ASG-XBLive are you using
3. What did you do?
4. What did you expect to see?
5. What did you see instead?
